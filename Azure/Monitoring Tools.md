## Azure Advisor
Evaluates your Azure resources and makes recommendations to help improve reliability, security and performance and reduce costs.

# Azure Service Health
Helps keep track of Azure resources, both your specifically deployed resources and the overall status of Azure.

# Azure Monitor
A platform for collecting data on your resources, analyzing that data, visualizing the information and event acting on the results.

# Azure Log Analytics
Lets you write and run log queries on the data gathered b Azure Monitor.

# Azure Monitor Alerts
An automated way to stay informed when Azure Monitor detects a threshold being crossed.
