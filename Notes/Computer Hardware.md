Physical parts of a computer system that work together to perform computing tasks.

**CPU**
The Central Processing Unit (CPU) is responsible for executing instructions and processing data.

**Motherboard**
The motherboard is the main circuit board that connects and allows communication between the CPU, memory, GPU, etc.

**RAM**
Random Access Memory (RAM) serves as the computers short-term memory, storing data that is actively being used by the CPU.

**Storage**
Hard Disk Drive (HDD) or Solid-State Drive (SSD) is where data is permanently stored (until the user deletes it). This includes the operating system, applications and files.

**Power Supply**
The Power Supply Unit (PSU) provides necessary electrical power to run the components.

**GPU**
The Graphical Processing Unit (GPU) is used for rendering images and videos on your screen. (Can have a dedicated graphics card or integrated graphics within the CPU).

**Peripherals**
Devices that are connected to the computer externally, such as keyboards, mice, monitors, etc.
