Used to detect and monitor issues related to the network performance of infrastructure as a service (Iaas) resource in Microsoft Azure.

Network watcher consists of 3 major sets of tools and capabilities:
- Monitoring
    - Topology
    - Connection monitor
- Network diagnostic
    - IP flow verify
    - NSG diagnostics
    - Next hop
    - Effective security rules
    - Connection troubleshoot
    - Packet capture
    - VPN troubleshoot
- Traffic
    - Flow logs
    - Traffic analytics
